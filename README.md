ResumeForge — Free Resume Builder
A free, beautiful resume builder with live preview and PDF download. Built with React + Vite.
Features
Live preview as you type
3 templates: Modern, Classic, Minimal
PDF download
Sections: Personal Info, Experience, Education, Skills, Projects
Fully responsive
Built by
Riddhima Gupta — guptariddhima75@gmail.com
Stack
React 18 + Vite
html2pdf.js for PDF export
Deployed on Vercel
Run locally
```bash
npm install
npm run dev
```
Deploy
Connect this repo to Vercel — it auto-detects Vite and deploys.
